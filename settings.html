The System Settings page provides a centralised location for configuring system preferences, managing account settings, and customising application behaviour.

It allows users and administrators to maintain consistency and control across the system.  

Overview

System Settings enable users to:

  • Configure personal and system preferences
  • Manage notifications and updates
  • Access support and help resources
  • Maintain account-related information

Search Controls

The search function allows users to quickly locate settings, pages or tools by entering keywords.

This improve navigation efficiency and helps users find configurations faster.
