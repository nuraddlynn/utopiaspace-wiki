This section provides solutions to common technical issues and system errors encountered by users.  

Follow the recommended steps to resolve problems before contacting support.

Login Issues

Problem: Unable to log in using Google account

Possible Causes:

  • Incorrect Google account
  • Account not registered in the system
  • Browser session issue

Solution:

  • Ensure you are using your registered company Google account
  • Log out and sign in again
  • Clear browser cache and cookies
  • Contact administrator if access is still denied

Page Not Loading

Problem: System page does not load or appears blank

Possible Causes:

  • Slow internet connection
  • Browser compatibility issues
  • Temporary system error

Solution:

  • Refresh the page
  • Check your internet connection
  • Try a different browser (Chrome recommended)
  • Wait and try again later

Missing Data or Modules

Problem: Unable to see certain modules or data

Possible Causes:

  • Role and permission restrictions
  • Incorrect filters applied

Solution:

  • Check your role and assigned access
  • Adjust filters (company, location, status)
  • Contact administrator if access is required

Taskboard Issues

Problem: Task not appearing on Taskboard

Possible Causes:

  • Task assigned to another user
  • Task status filter hiding it

Solution:

  • Check task status (To-do, Done, KIV)
  • Use search function to locate task
  • Verify assigned user

Calendar issues

Problem: Events or leave not visible

Possible Causes:

  • Filters are enabled
  • Data not updated

Solution:

  • Check filter settings (company, location, toggles)
  • Refresh the page
  • Select correct date

Meeting Notes Issues

Problem: Unable to save meeting notes

Possible Causes:

  • Required fields not filled
  • System validation error

Solution:

  • Ensure all required fields are completed
  • Try saving again
  • Refresh page if issue persists

Payment / Approval Issues

Problem: Unable to approve or process requests

Possible Causes:

  • Insufficient permissions
  • Request not in correct stage

Solution:

  • Confirm your role allows approval
  • Check request status
  • Contact administrator if necessary

Performance Issues

Problem: System is slow or unresponsive

Possible Causes:

  • High system usage
  • Network usability

Solution:

  • Refresh the page
  • Close unused tabs
  • Check your internet connection
  • Try again later

When to Contact Support

If the issue persists after trying the above solutions:

  • Contact the Support Center
  • Submit an Incident Report
  • Provide screenshots and details for faster resolution
