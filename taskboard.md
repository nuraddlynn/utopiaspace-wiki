## Overview

<div class="utopia-taskboard-grid">
  <section>
    <h3>Create Tasks</h3>
    <p>Add tasks with titles, descriptions, assignees, priorities, due dates, and attachments.</p>
  </section>

  <section>
    <h3>Track Progress</h3>
    <p>Move tasks between board columns so work status stays visible and current.</p>
  </section>

  <section>
    <h3>Manage Workload</h3>
    <p>Use search, filters, priorities, and deadlines to monitor productivity and avoid missed tasks.</p>
  </section>
</div>

## Board Layout

<div class="utopia-taskboard-table">
  <div class="utopia-taskboard-head">Column</div>
  <div class="utopia-taskboard-head">Meaning</div>

  <strong>To-Do</strong>
  <p>Tasks that have been created but have not started yet.</p>

  <strong>Done</strong>
  <p>Tasks that have been completed.</p>

  <strong>KIV</strong>
  <p>Tasks that are pending, paused, or kept in view for later follow-up.</p>
</div>

## Creating a Task

<div class="utopia-taskboard-flow">
  <div>
    <strong>1</strong>
    <span>Open Taskboard from the sidebar.</span>
  </div>
  <div>
    <strong>2</strong>
    <span>Click Create Task to start a new task.</span>
  </div>
  <div>
    <strong>3</strong>
    <span>Fill in the task title, description, assignee, priority, and due date.</span>
  </div>
  <div>
    <strong>4</strong>
    <span>Upload supporting attachments if the task requires reference files.</span>
  </div>
  <div>
    <strong>5</strong>
    <span>Click Submit to create the task.</span>
  </div>
</div>

## Task Details

<div class="utopia-taskboard-table">
  <div class="utopia-taskboard-head">Field</div>
  <div class="utopia-taskboard-head">Purpose</div>

  <strong>Title</strong>
  <p>Short task name that explains the work to be done.</p>

  <strong>Description</strong>
  <p>Detailed task information, requirements, or notes for the assignee.</p>

  <strong>Assign To</strong>
  <p>Employee responsible for completing the task.</p>

  <strong>Priority</strong>
  <p>Urgency level, such as Low, Medium, or High.</p>

  <strong>Due Date</strong>
  <p>Deadline for task completion.</p>

  <strong>Attachments</strong>
  <p>Supporting files or references needed to complete the task.</p>
</div>

## Managing Tasks

<div class="utopia-taskboard-table">
  <div class="utopia-taskboard-head">Action</div>
  <div class="utopia-taskboard-head">When to use it</div>

  <strong>Move Task</strong>
  <p>Move a task between columns when the status changes.</p>

  <strong>Update Details</strong>
  <p>Edit task information when requirements, priority, or due date changes.</p>

  <strong>Reassign Task</strong>
  <p>Change the assignee when ownership moves to another employee.</p>

  <strong>Search and Filter</strong>
  <p>Find specific tasks or narrow the board by criteria.</p>

  <strong>Delete Task</strong>
  <p>Remove a task when it is no longer needed.</p>
</div>

## Tips

<div class="utopia-taskboard-grid">
  <section>
    <h3>Assign Clearly</h3>

  </section>

  <section>
    <h3>Use Priority</h3>
    <p>Set priority levels to make urgent work easier to identify.</p>
  </section>

  <section>
    <h3>Update Often</h3>
    <p>Move tasks regularly so the board reflects current progress.</p>
  </section>
</div>
