The Taskboard is a visual task management tool that helps users
organise, track, and manage tasks efficiently using a board layout.

Tasks are grouped into different stages, allowing users to monitor
progress and ensure completion.  

# Overview

With the Taskboard, users can:

- Create and assign tasks
- Track task progress using a board layout
- Manage task priorities and deadlines
- Upload attachments for tasks
- Monitor team productivity

# Taskboard Layout

The Taskboard is divided into columns representing task status:

- To-Do - Tasks that have not been started
- Done - Completed tasks
- KIV (Keep In View) - Tasks that are pending or on hold

# Creating a Task

To create a new task:

1.  Navigate to *Taskboard*
2.  Click *+ Create Task*
3.  Fill in the required details:

## Task Information

- Title - Enter the task name
- Description - Provide task details

## Assignment

- Assign To - Select the employee responsible for the task

## Task Settings

- Priority - Select priority level (e.g., Low, Medium, High)
- Due Date - Set the deadline for task completion

## Attachments

Upload supporting files if necessary

4\. Click *Submit* to create the task

# Managing Tasks

Users can:

- Move tasks between columns (e.g., To-do ⇢ Done)
- Update task details
- Reassign tasks
- Edit priority and due date
- Delete tasks if no longer needed

# Searching and Filtering

- Use the Search Bar to find specific tasks
- Use Filters to refine task results based on criteria

# Task Tracking

- Tasks are visually tracked based on their status
- Users can quickly identify pending, completed, or delayed tasks
- Progress is updated in real-time

# Tips

- Assign tasks clearly to avoid confusion
- Set realistic deadlines
- Use priority levels to manage urgency
- Move tasks regularly to reflect current progress
