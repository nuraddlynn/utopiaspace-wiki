User Management refers to the process of maintaining user accounts and ensuring that user information within the system is accurate and up to date.

Although there is no dedicated module, user management is handled through administrative controls and existing system.  

Overview

Administrators are responsible for managing user-related information across the system.

This includes:

  • Creating user accounts
  • Updating user details
  • Managing user status
  • Maintaining accurate records

User Information

Each user in the system typically includes:

  • Personal Information (e.g., Name, Email Address, Date of Birthdate, Gender, Nationality, Religion, etc)
  • Team Information (e.g., Company, Business Units, Departments, Position, etc)
  • Employment Details (e.g., Employment Type, Working Shift, Date, etc)
  • Contact & Emergency (e.g., Phone, Email, Work Email, Personal Email, Emergency Contact, Relationship, etc)
  • Bank Information (e.g., Bank Holder, Bank Name, Account Number, etc)

Managing Users

User management activities may include:

  • Adding new users to the system
  • Updating user profile information
  • Deactivating users who are no longer active
  • Ensuring all user records are accurate and consistent

Key Responsibilities

Administrators should:

  • Ensure all users are valid and properly registered
  • Keep user information updated at all times
  • Remove or deactivate inactive users
  • Maintain consistency across all user records
